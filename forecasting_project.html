TÜİK Forecasting Project: Health Expenditures

Burak Kara

1. Project Overview

This project develops an R-based forecasting analysis using a time series data set obtained from the TÜİK Data Portal.

The selected data set is Health Expenditures under the Health and Social Protection theme. The selected variable is Total Health Expenditure - General Total, measured in million TRY.

The purpose of the project is to forecast the next available annual period after the latest TÜİK observation.

7. Forecast Accuracy Function

calculate_accuracy <- function(actual, forecast) {
  error <- actual - forecast
  mad <- mean(abs(error), na.rm = TRUE)
  rsfe <- sum(error, na.rm = TRUE)
  
  tibble(
    Bias = mean(error, na.rm = TRUE),
    MAD = mad,
    MSE = mean(error^2, na.rm = TRUE),
    MAPE = mean(abs(error / actual), na.rm = TRUE) * 100,
    RSFE = rsfe,
    Tracking_Signal = rsfe / mad
  )
}

8. Forecasting Methods

The following applicable methods are used:

  • Naïve Forecasting
  • Moving Average
  • Weighted Moving Average
  • Exponential Smoothing
  • Trend-Adjusted Exponential Smoothing
  • Linear Trend Projection

Since the selected data are annual, the following methods are not applicable:

  • Seasonal Indices
  • Additive Decomposition
  • Multiplicative Decomposition
  • Regression with Trend and Seasonal Dummy Variables
years <- health_expenditure_annual$year
values <- health_expenditure_annual$health_expenditure_million_try

min_train <- 10
eval_index <- (min_train + 1):length(values)

actual_eval <- values[eval_index]
year_eval <- years[eval_index]

forecast_naive <- c()
forecast_ma <- c()
forecast_wma <- c()
forecast_ses <- c()
forecast_holt <- c()
forecast_trend <- c()

for (i in eval_index) {
  train_values <- values[1:(i - 1)]
  train_ts <- ts(train_values, start = years[1], frequency = 1)
  
  forecast_naive <- c(forecast_naive, tail(train_values, 1))
  
  forecast_ma <- c(forecast_ma, mean(tail(train_values, 3)))
  
  forecast_wma <- c(
    forecast_wma,
    sum(tail(train_values, 3) * c(0.2, 0.3, 0.5))
  )
  
  forecast_ses <- c(
    forecast_ses,
    as.numeric(forecast::ses(train_ts, h = 1)$mean[1])
  )
  
  forecast_holt <- c(
    forecast_holt,
    as.numeric(forecast::holt(train_ts, h = 1)$mean[1])
  )
  
  trend_df <- data.frame(
    y = train_values,
    t = seq_along(train_values)
  )
  
  trend_model <- lm(y ~ t, data = trend_df)
  
  forecast_trend <- c(
    forecast_trend,
    as.numeric(predict(trend_model, newdata = data.frame(t = length(train_values) + 1)))
  )
}

forecast_results <- tibble(

11. Actual vs Forecast Plots

plot_forecast_method <- function(method_column, method_name, filename) {
  plot_data <- forecast_results %>%
    select(year, actual, forecast = all_of(method_column)) %>%
    tidyr::pivot_longer(
      cols = c(actual, forecast),
      names_to = "series",
      values_to = "value"
    )
  
  p <- ggplot(plot_data, aes(x = year, y = value, linetype = series)) +
    geom_line(linewidth = 1) +
    geom_point() +
    scale_y_continuous(labels = comma) +
    labs(
      title = paste("Actual vs Forecast:", method_name),
      x = "Year",
      y = "Million TRY",
      linetype = "Series"
    )
  
  ggsave(
    filename = paste0("outputs/figures/", filename),
    plot = p,
    width = 8,
    height = 5
  )
  
  p
}

14. Interpretation of Results

The selected TÜİK variable represents total health expenditure in Türkiye, measured in million TRY. The series is suitable for forecasting because it is numerical, ordered by year, and observed repeatedly over a long time period.

The time series displays a strong upward trend. This is expected because health expenditures tend to increase over time due to price increases, population dynamics, service expansion, and broader economic growth. Since the data are annual, there is no monthly or quarterly seasonal pattern to model.

The superior method was selected by comparing the accuracy measures of the applicable forecasting methods. The final forecast represents the estimated total health expenditure for 2025, based on the latest available TÜİK observation for 2024.

15. Limitations

This analysis has several limitations. First, the data are annual, so the number of observations is limited compared with monthly data. Second, the series has a strong nominal upward trend, which may be affected by inflation and structural changes. Third, the analysis uses only historical values of health expenditures and does not include explanatory variables such as inflation, population, exchange rates, or policy changes. Finally, TÜİK data may be revised in future releases.
